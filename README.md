# ST7735-LCD
ST7735 128X128 SPI Display Code &amp; Projects



### Wiring For 5110 to Arduino Nano
| Pin On LCD | Pin on NANO |
| ---------- |----------------|
| LED  | 3.3 or 5V depending on your display  |
| SCK  | D13 |
| SDA  | D11 (MOSI) |
| A0   | D8 |
| RST  | D9  |
| CS   | D10 |
|GND   | GND |
